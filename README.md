# Traveling App

Next.js додаток із завданнями з пошуку турів (країни, ціни, готелі).

## Запуск локально

### Вимоги

- Node.js 18+
- npm, pnpm, yarn або bun

### Кроки

1. Клонуйте репозиторій та перейдіть у папку проєкту:

```bash
git clone <url-репозиторію>
cd traveling-app
```

2. Встановіть залежності:

```bash
npm install
```

3. Запустіть dev-сервер:

```bash
npm run dev
```

4. Відкрийте в браузері: [http://localhost:3000](http://localhost:3000).

### Скрипти

| Команда        | Опис                           |
|----------------|---------------------------------|
| `npm run dev`  | Запуск у режимі розробки        |
| `npm run build`| Збірка для production          |
| `npm run start`| Запуск production-збірки      |
| `npm run lint` | Перевірка коду (ESLint)        |
